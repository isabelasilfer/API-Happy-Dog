# Doguineo 🐶

Script em Python que consome a API pública dog.ceo para obter uma imagem aleatória de cachorro.

## Funcionalidades

- Requisição HTTP utilizando requests
- Conversão da resposta para JSON
- Extração da URL da imagem
- Abertura automática no navegador padrão

## Tecnologias utilizadas

- Python
- requests
- webbrowser
